创建管理员

创建管理员

  • 管理员不是由页面上注册得来的，而是由运维人员创建的
  • 管理员也是用户，只是不是普通用户，在用户表中使用isAdmin标识是否为管理员
  • 创建super_command.py文件，用于编写应用程序的扩展命令
  • 定义命令类SuperCommand，用于创建管理员
import random
from datetime import datetime
from flask import current_app
from flask_script.commands import Command
from models import db, UserInfo


class SuperCommand(Command):
    def run(self):
        print('请输入用户名：')
        mobile = input()
        print('请输入密码：')
        pwd = input()

        user = UserInfo()
        user.mobile = mobile
        user.password = pwd
        user.isAdmin = True
        db.session.add(user)
        db.session.commit()
        print('创建管理员成功')
  • 在project.py中添加命令
from super_command import SuperCommand
manager.add_command('createsuper', SuperCommand())
  • 在Terminal窗口中执行命令，创建管理员，按提示填写用户名、密码
python project.py createsuper
